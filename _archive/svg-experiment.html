SVG Experiment

min-x: 0 / min-y: 0

Die Dimension (width/height) des SVG bestimmt 1:1 die Grösse der Viewbox, wie sie im Documentflow gezeichnet wird. Width und Height der Viewbox selber haben dabei keinen Einfluss. Das ist zunächst kontraintuitiv.

Das Verhältnis von SVG-Dimensionen zu Viewbox-Dimensionen wirkt wie ein Zoom-Faktor auf die Elemente im SVG. Zoom-Faktor = SVG-Dimension/Viewbox-Dimension.
